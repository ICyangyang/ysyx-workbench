## 命令行操作
- 查找名为.git的文件夹：
find /path/to/search -type d -name .git
或者
ls -la | grep .git
> /path/to/search替换为你想要开始搜索的目录的路径
> 只想在当前用户的主目录中搜索，可使用~
> 要在当前目录及其所有子目录中搜索，可以使用：find . -type d -name .git
